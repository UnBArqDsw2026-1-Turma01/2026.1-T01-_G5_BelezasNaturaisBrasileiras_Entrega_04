## Sobre o Projeto

A **Plataforma de Belezas Naturais e Trilhas** é um ecossistema digital focado no mapeamento colaborativo de pontos turísticos naturais no Brasil. O sistema permite que visitantes explorem cachoeiras, montanhas e trilhas através de um feed inteligente com filtros por Estado, Cidade e Região.

O projeto visa solucionar a fragmentação de informações e a falta de segurança no agendamento de trilhas, conectando usuários a organizadores através de chats individuais, sistemas de validação de presença via código único e badges de conquista. A arquitetura do sistema prevê diferentes níveis de acesso (Visitante, Usuário Comum, Organizador e Administrador), garantindo a integridade dos dados através de histórico de edições e moderação ativa.

## Informações Complementares

- **Backend:** Desenvolvido em NestJS (RNF01).
- **Frontend:** Desenvolvido em React com Vite e TypeScript (RNF02).
- **Banco de Dados:** PostgreSQL (RNF03).
- **Regras de Negócio:** O sistema implementa versionamento de edições (RN01) e restrição de chat 1-para-1 (RN03) para garantir a segurança dos usuários.
