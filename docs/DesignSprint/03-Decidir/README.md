# Etapa 3 - Decidir

A etapa **Decidir** seleciona as ideias com maior potencial.
O time avalia as propostas, discute criterios e define o caminho que sera prototipado.

## Objetivo

Tomar decisoes claras sobre a solucao que avancara para prototipacao.

## Atividades da etapa

- apresentacao dos esbocos
- votacao estruturada das alternativas
- analise de viabilidade e impacto
- definicao da proposta final

## Entregaveis esperados

- registro da decisao da equipe
- justificativa da alternativa escolhida
- escopo do que sera prototipado
