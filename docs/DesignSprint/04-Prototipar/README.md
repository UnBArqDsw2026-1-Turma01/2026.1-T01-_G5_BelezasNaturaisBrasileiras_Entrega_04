# Etapa 4 - Prototipar

A etapa **Prototipar** materializa a solucao escolhida em uma representacao testavel.
O foco e construir algo suficiente para validar hipoteses com usuarios reais.

## Objetivo

Criar um prototipo realista, rapido e funcional para o contexto do teste.

## Atividades da etapa

- definicao do escopo do prototipo
- distribuicao de responsabilidades
- construcao colaborativa do prototipo
- revisao final para preparo do teste

## Entregaveis esperados

- prototipo navegavel ou representativo
- roteiro de uso para a etapa de teste
