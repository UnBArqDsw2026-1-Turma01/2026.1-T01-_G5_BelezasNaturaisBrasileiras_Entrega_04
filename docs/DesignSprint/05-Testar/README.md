# Etapa 5 - Testar

A etapa **Testar** valida o prototipo com usuarios e coleta evidencias para orientar os proximos passos.
O objetivo e aprender com feedback real para reduzir riscos da solucao.

## Objetivo

Verificar se a proposta atende as necessidades do usuario e identificar melhorias.

## Atividades da etapa

- preparo das sessoes de teste
- conducao de entrevistas e observacao de uso
- registro de percepcoes e dificuldades
- consolidacao dos aprendizados

## Entregaveis esperados

- relatorio de testes com principais achados
- lista de melhorias e proximas acoes
