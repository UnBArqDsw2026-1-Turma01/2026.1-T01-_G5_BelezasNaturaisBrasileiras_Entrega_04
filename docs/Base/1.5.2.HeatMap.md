# Tabela de Disponibilidade

Para organizar melhor o andamento do projeto, a equipe elaborou uma **Tabela de Disponibilidade** com os horários em que cada integrante está apto a contribuir.  
Esse recurso foi fundamental para planejar reuniões, dividir tarefas de forma equilibrada e otimizar a colaboração entre os membros.

## Reuniões

No quesito de reuniões síncronas, a parte das reuniões em sala de aula quando possível, foi decidido:

- Reunião principal na segunda-feira, das 20:00 às 21:00.
- Reunião secundária no sábado, entre as 15:00 e 17:00.

Foi decidido também pela equipe em fazer reuniões assíncronas, através das discussões feitas no grupo de WhatsApp, tendo as atas separadas das de reuniões síncronas.
