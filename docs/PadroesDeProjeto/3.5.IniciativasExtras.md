# 3.5. Iniciativas Extras (Padrões de Projeto)

Além dos padrões de projeto exigidos na entrega, a equipe realizou duas iniciativas extras que complementam e evidenciam a aplicação dos padrões GoF no projeto Belezas Naturais Brasileiras.

## Frontend — Aplicação Web React

A equipe desenvolveu um frontend completo em React 18 + TypeScript + Vite + TailwindCSS para consumir a API do backend. A aplicação possui 11 páginas, controle de acesso por roles, contexto de autenticação com expiração de sessão e módulos de API por domínio.

→ [3.5.1. Frontend — Aplicação Web React](IniciativasExtras/3.5.1.Frontend.md)

## Testes Automatizados e de Integração

A equipe construiu uma suíte de **23 arquivos de teste Jest** cobrindo todos os 18 padrões GoF implementados, além de um **script de integração end-to-end com 40 passos** que exercita o fluxo completo do sistema e um guia de testes manuais com `curl`.

→ [3.5.2. Testes Automatizados e de Integração](IniciativasExtras/3.5.2.Testes.md)
