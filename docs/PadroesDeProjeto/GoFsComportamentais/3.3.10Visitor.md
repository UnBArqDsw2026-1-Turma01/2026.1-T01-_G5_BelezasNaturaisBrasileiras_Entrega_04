## Introdução

O **Visitor** é um padrão comportamental que representa uma operação a ser realizada nos elementos de uma estrutura de objetos, deixando que você defina uma nova operação sem mudar as classes dos elementos. É útil quando você deseja adicionar novos comportamentos a uma estrutura de classes sem modificá-las.

Este padrão separa algoritmos da estrutura de dados, permitindo adicionar novas operações facilmente.

## Quando Aplicar?

- Quando você possui uma estrutura de objetos e deseja realizar operações neles sem acoplamento
- Quando múltiplas operações diferentes e não relacionadas são necessárias em objetos
- Quando você quer evitar modificar as classes estruturais quando novas operações são adicionadas
- Quando você precisa adicionar comportamentos específicos a diferentes tipos de elementos
- Quando muitas operações distintas precisam ser executadas em uma estrutura de classes
