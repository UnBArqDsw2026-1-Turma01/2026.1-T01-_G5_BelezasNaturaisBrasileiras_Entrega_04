## Introdução

O **Interpreter** é um padrão comportamental que define uma representação gramatical para uma linguagem e um interpretador para processar sentenças dessa linguagem. É útil quando você precisa interpretar e executar uma linguagem ou um conjunto de regras específicas.

Este padrão permite criar linguagens customizadas e interpretadores para elas sem modificar o código existente.

## Quando Aplicar?

- Quando você precisa interpretar uma linguagem ou conjunto de regras
- Quando a linguagem é relativamente simples e pode ser representada em uma gramática
- Quando desempenho não é crítico (interpretadores são geralmente lentos)
- Quando você deseja criar uma DSL (Domain Specific Language)
- Quando múltiplas interpretações da mesma entrada são possíveis
