## Introdução

O **State** é um padrão comportamental que permite que um objeto altere seu comportamento quando seu estado interno muda, parecendo mudar a classe do objeto. É útil quando o comportamento de um objeto deve variar dependendo de seu estado.

Este padrão encapsula diferentes comportamentos em classes de estado separadas, permitindo que o objeto delegue o comportamento apropriado.

## Quando Aplicar?

- Quando o comportamento de um objeto depende do seu estado e deve mudar em tempo de execução
- Quando operações têm grande condicionalidade baseada no estado
- Quando você deseja eliminar grandes instruções condicionais
- Quando múltiplos objetos podem ter estados similares
- Quando o código tem muitas instruções switch ou if baseadas em estado
