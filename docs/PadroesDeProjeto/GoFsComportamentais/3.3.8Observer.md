## Introdução

O **Observer** é um padrão comportamental que define uma dependência um-para-muitos entre objetos, de modo que quando um objeto muda de estado, todos os seus dependentes são notificados automaticamente. É útil quando múltiplos objetos precisam reagir a mudanças de estado.

Este padrão desacopla produtores de notificações de seus consumidores, permitindo que qualquer número de observadores se registrem para receber notificações.

## Quando Aplicar?

- Quando uma mudança em um objeto requer alterar outros objetos, mas você não sabe quantos
- Quando um objeto deve notificar outros sem assumir quem são esses objetos
- Quando você precisa construir um sistema de notificações flexível
- Quando você deseja implementar um padrão publicador-assinante
- Quando múltiplas partes da aplicação precisam reagir aos mesmos eventos
