## Introdução

O **Mediator** é um padrão comportamental que define um objeto que encapsula como um conjunto de objetos interage, promovendo acoplamento fraco. É útil quando múltiplos objetos precisam se comunicar de forma controlada e coordenada.

Este padrão centraliza a lógica complexa de comunicação em um mediador, permitindo que os objetos participantes se concentrem em suas responsabilidades.

## Quando Aplicar?

- Quando um conjunto de objetos deve se comunicar de forma controlada
- Quando a lógica de interação entre objetos é complexa
- Quando múltiplos objetos referenciam uns aos outros de forma desordenada
- Quando comportamentos distribuídos entre vários objetos devem ser centralizados
- Quando você quer reutilizar objetos que estão acoplados a muitos outros
