## Introdução

O **Prototype** é um padrão criacional que permite criar novos objetos clonando um objeto protótipo, em vez de criá-los do zero. É particularmente útil quando a criação de um objeto é cara, complexa ou quando você deseja evitar a dependência de classes concretas.

Este padrão define uma interface para clonar um objeto, permitindo que diferentes tipos de objetos sejam duplicados sem conhecer seus detalhes internos de construção.

## Quando Aplicar?

- Quando criar uma instância é custoso e você pode economizar clonando um objeto já existente
- Quando há muitas subclasses e você quer evitar uma hierarquia de classes de criação
- Quando a configuração inicial de um objeto é complexa
- Quando você precisa de cópias independentes de objetos com estado pré-configurado
- Quando deseja implementar um sistema de undo/redo através de snapshots de estado
