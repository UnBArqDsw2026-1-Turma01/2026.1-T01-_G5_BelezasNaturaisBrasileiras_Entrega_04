## Introdução

O **Flyweight** é um padrão estrutural que usa compartilhamento para sustentar eficientemente grandes quantidades de objetos finos, economizando memória. É útil quando você tem muitos objetos similares que consomem grandes quantidades de memória.

Este padrão separa o estado intrínseco (compartilhado) do estado extrínseco (específico de cada objeto), permitindo reutilizar instâncias.

## Quando Aplicar?

- Quando um programa cria muitos objetos similares
- Quando os custos de armazenamento são altos devido ao número de objetos
- Quando você pode compartilhar a maior parte do estado de um objeto
- Quando muitos objetos podem ser substituídos por menos objetos compartilhados
- Quando a aplicação não depende da identidade do objeto
