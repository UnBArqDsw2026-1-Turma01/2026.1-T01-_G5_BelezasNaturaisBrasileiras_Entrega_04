## Introdução

O **Facade** é um padrão estrutural que fornece uma interface unificada e simplificada para um conjunto de interfaces em um subsistema. É útil quando você deseja simplificar a complexidade de interação com múltiplos componentes.

Este padrão atua como uma fachada, ocultando a complexidade interna de um subsistema e fornecendo uma interface simples para os clientes.

## Quando Aplicar?

- Quando você deseja fornecer uma interface simples para um subsistema complexo
- Quando há muitas dependências entre clientes e classes de implementação
- Quando você deseja estruturar subsistemas em camadas
- Quando deseja desacoplar clientes de componentes de um subsistema
- Quando múltiplas classes relacionadas precisam ser usadas juntas
